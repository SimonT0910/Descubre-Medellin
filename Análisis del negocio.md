¿Quién usa el sistema?

El sistema cuenta con un único tipo de usuario: el usuario registrado, quien puede registrarse, iniciar sesión, consultar, buscar y filtrar lugares, visualizar sus detalles, publicar nuevos lugares, comentar, calificar, guardar favoritos y gestionar su perfil. No existen roles adicionales en esta versión del sistema.

¿Qué necesita hacer?

-	Buscar lugares por nombre o descripción.
-	Filtrar lugares por categoría o rango de precio.
-	Ver la lista de lugares disponibles.
-	Consultar el detalle completo de un lugar.
-	Registrarse en la plataforma.
-	Iniciar sesión.
-	Publicar nuevos lugares.
-	Calificar y comentar lugares.
-	Guardar lugares como favoritos.
-	Ver su perfil y sus lugares favoritos.

¿Qué datos maneja?

Datos de usuarios:
-	Nombre.
-	Correo electrónico.
-	Contraseña.

Datos de lugares:
-	Nombre del lugar.
-	Descripción.
-	Categoría.
-	Precio aproximado.
-	Ubicación.
-	Usuario que lo público.

Datos de interacción:
-	Comentarios.
-	Calificaciones.
-	Favoritos.

¿Qué restricciones existen?
-	Solo los usuarios registrados pueden comentar, calificar o publicar lugares.
-	Un usuario debe iniciar sesión para guardar favoritos.
-	No se pueden registrar dos usuarios con el mismo correo electrónico.
-	Los comentarios deben estar asociados a un lugar existente.
-	Un lugar solo puede agregarse una vez a favoritos por usuario.
-	El sistema debe validar los datos antes de guardarlos en la base de datos.
